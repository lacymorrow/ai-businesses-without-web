# AI Business Website Opportunity Finder

## Project Overview
A marketing analytics dashboard that identifies businesses without proper websites by analyzing Google Maps data. The system will help identify potential clients who could benefit from web development services.

## Data Model

### Business
```typescript
interface Business {
  id: string;
  name: string;
  address: string;
  location: {
    lat: number;
    lng: number;
  };
  category: string[];
  websiteType: 'none' | 'facebook' | 'yelp' | 'other';
  websiteUrl?: string;
  socialProfiles: {
    facebook?: string;
    yelp?: string;
    instagram?: string;
    other?: string[];
  };
  phoneNumber?: string;
  lastUpdated: Date;
  searchQuery: string;
}
```

### Search Parameters
```typescript
interface SearchParameters {
  location: string;
  radius: number;
  categories?: string[];
  excludeWebsiteTypes?: ('facebook' | 'yelp' | 'other')[];
  limit?: number;
  sortBy?: 'distance' | 'relevance' | 'rating';
}
```
